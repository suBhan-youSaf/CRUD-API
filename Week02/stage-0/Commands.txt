Commands:
npm init -y         (It instantly creates a file in your directory called package.json. This file is a configuration 		    manifest. It tracks your project's name, version, and most importantly, a list of every third-   		    party package your app relies on to run  && it creates project)
npm install express swagger-ui-express (it fetches the tools you need to set up the server && It creates a massive 					folder in your project called node_modules where it dumps the downloaded 					code, and it records both tools inside your package.json file so your app 					knows they are ready to use.)
Notes:
express - This fetches the Express framework. It provides the functions that let you spin up a server on port 3000, listen for incoming web requests, read JSON payloads, and send back status codes like 200 or 404. Without it, you'd have to write hundreds of lines of low-level networking code by hand.

swagger-ui-express: This fetches the tool that reads your openapi.json file and automatically builds that beautiful, interactive documentation page at /docs with the "Try it out" buttons.

npm - node package manager

-y stands for (set all to yes for default)
